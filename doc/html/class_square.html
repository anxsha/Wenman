Detailed Description

Every object of this class contains information about one assigned graphical square. It is the source, based on which graphical grid is updated.

Constructor & Destructor Documentation

◆ Square()

Square::Square ( const std::vector< Square > &  v)
explicit

This constructor sets the number of all animals on this square to 0. Receives the Game's vector which consists of all squares as a parameter to identify its index therein.
